## ⚡ CONCEPT
**K-Crossing** este o platformă de e-commerce de tip Single Page Application (SPA) dedicată culturii streetwear. Proiectul îmbină o estetică industrială brută cu o arhitectură frontend ultra-performantă, oferind o experiență de utilizator fluidă și vizuală.

> **"NO LIMITS. JUST STYLE."**

---

## 🚀 CARACTERISTICI CHEIE
* **Design Industrial & Monocromatic:** Interfață minimalistă bazată pe contrast, tipografie bold și animații Glassmorphism.
* **Navigare cu Auto-Scroll:** Filtrele din Navbar (Noutăți, Reduceri) declanșează automat scroll-ul către secțiunea de produse.
* **Shopping Experience:** Sistem de Wishlist integrat și filtrare dinamică după gen (Men/Women/Unisex).
* **Checkout Funcțional:** Integrare cu **Stripe API** pentru procesarea securizată a plăților online.
* **Contact Direct:** Formular de suport integrat cu **EmailJS**, trimițând notificări formatate HTML direct către echipa de admin.

---

## 🛠️ TECH STACK
* **Core:** [React.js](https://reactjs.org/) + [Vite](https://vitejs.dev/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Animații:** [Framer Motion](https://www.framer.com/motion/)
* **Icons:** [Lucide React](https://lucide.dev/)
* **Backend Services:** EmailJS (Support), Stripe (Payments)
* **Hosting:** [Vercel](https://vercel.com/)
